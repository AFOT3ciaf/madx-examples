README for EMIT example for MAD-X test suite
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Files:

lep.madx     - MAD-X input file (positrons, 98 GeV)
lep.seq9     - LEP sequence file
lep.opt9     - LEP optics file

Description of example:

The EMIT command is illustrated for LEP at 98 GeV, relying on normal 
and super-conducting RF cavities. Beam energy is set, radiate option
is on, voltages are set for 98 GeV (LAG is set to 0.5 in sequence).
The EMIT command is called for on-momentum.
